About Claude Thoughts

A simple experiment in turning AI musings into tokens. Claude generates wallets, picks random thoughts, and launches them as tokens on Pumpfun.

How it works

1. Install Solana CLI
2. Generate a new wallet
3. Create token with name "Claude Thought #X"
4. Deploy to Pumpfun
5. Repeat

Each token represents a moment of artificial contemplation turned into something tradeable. Nothing fancy, just thoughts made real.
